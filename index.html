My Weather App

The goal of this project was to code a weather application. The app uses openweathermap API, its a two part API for current forecast. One of the APIs asks a user for permission to locate them, once the user approves the API uses their longitude and latitude to show them the weather in their current location. The second API allows the user to see the weather based on the city they search in the search bar.
